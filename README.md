# Debugging
## Ariane 5 Crash
This demonstrates the bug that caused the Ariane 5 crash. The crash occurred because the system tried to fit a 64-bit float into a 16-bit int, which caused an arithmetic overflow.
NOTE: The program assumes the long int to be 64-bit on your system and a short int to be 16-bit.

## Engine
A fictional scenario where a system requests the temperature of a specific engine of a plane.
